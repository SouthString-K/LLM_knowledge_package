## T-RO 论文职责与创新点

### 负责部分

- 负责 T-RO 腹腔镜相机自主控制论文中的视觉算法部分
- 负责检测、细粒度分析与角度变化感知相关视觉链路设计

### 视觉算法创新

| 模块 | 创新点 |
|------|------|
| **先验检测** | 使用自我微调的 **YOLOv26** 作为先验知识提取器 |
| **训练策略** | 针对 **多类别、多曝光、多环境** 数据进行微调训练，提升复杂手术视觉场景下的鲁棒性 |
| **细粒度分析** | 在 YOLOv26 检测结果基础上接入 **SAM2** 做细粒度区域分析 |
| **角度估计** | 使用 **凸包算法** 进行角度运算，精准检测角度变化 |
| **系统性能** | 视觉链路整体帧率可达到 **30 fps** |

### 架构理解

- **YOLOv26**：负责快速、稳定地提供目标级先验定位
- **SAM2**：负责基于先验检测结果进行更细粒度的结构分析
- **凸包角度计算**：负责从细粒度轮廓中提取稳定的角度变化信息

这套框架的核心价值是把**高速检测、精细分割、几何角度分析**串成一个统一视觉系统，既保证实时性，又提升了复杂场景下的精度与可解释性。
