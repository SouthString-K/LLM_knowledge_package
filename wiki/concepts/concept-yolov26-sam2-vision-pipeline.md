# YOLOv26 + SAM2 视觉链路

## 概念定义

这是一套面向复杂视觉场景的检测-分割-几何分析一体化视觉链路。其核心思路是先用自我微调的 YOLOv26 提供高速、稳定的目标级先验，再用 SAM2 做细粒度区域分析，最后通过凸包算法进行角度运算，实现对角度变化的精准感知。

## 架构组成

| 模块 | 作用 |
|------|------|
| **YOLOv26（微调）** | 负责目标级检测与先验知识提取 |
| **SAM2** | 负责基于检测先验的细粒度分割与结构分析 |
| **凸包算法** | 负责从轮廓几何中提取角度变化信息 |

## 训练与适应性

这套链路中的 YOLOv26 经过针对性微调，训练数据覆盖：

- 多类别
- 多曝光
- 多环境

这样做的目的，是提升复杂真实场景下的鲁棒性和先验检测稳定性。

## 方法亮点

1. 将高速检测与细粒度分割有效串联，而不是孤立使用检测或分割模型
2. 用微调后的 YOLOv26 作为先验知识，降低后续细粒度分析的不稳定性
3. 用 SAM2 在目标级检测基础上补足边界、局部结构和区域细节
4. 用凸包算法显式建模角度变化，增强几何可解释性
5. 整体视觉链路帧率可达到 **30 fps**

## 适用价值

| 价值 | 说明 |
|------|------|
| **实时性** | 能满足在线视觉分析需求 |
| **精细性** | 能从目标级定位走向区域级细节理解 |
| **可解释性** | 角度变化由几何算法显式计算，不完全依赖黑箱模型 |
| **鲁棒性** | 在多类别、多曝光、多环境训练下具备更强泛化能力 |

## 在 T-RO 论文中的作用

这套视觉链路是主人在 T-RO 腹腔镜相机自主控制工作中负责的核心视觉算法模块，支撑了：

- 快速目标先验定位
- 细粒度结构分析
- 稳定角度变化检测
- 实时视觉处理
